## ⚙️ Setup

1. Extract project ke folder `htdocs/threadb2b/`
2. Import database `database/_threadb2b.sql` ke phpMyAdmin.
3. Konfigurasi koneksi database pada file konfigurasi.
4. Jalankan Apache & MySQL melalui XAMPP.
5. Akses aplikasi melalui browser.

```
http://localhost/threadb2b/
```
